# Options_Algorithms_Trading
Algorithm trading for options put and call decision

### Learning Options
# _期权价格 = 内在价值 + 时间价值_

## **核心：期权价格由“定价模型”决定，不是由“买卖人数”决定**

期权市场有做市商和套利者，他们使用期权定价模型（如Black-Scholes）来报价。模型中最重要的五个参数是：
```
股价（S）
行权价（K）
剩余时间（T）
无风险利率（r）
隐含波动率（σ） ← 这是关键
当波动率（σ）上升时，模型输出的所有期权价格（无论看涨看跌）都会上升。
```
这不是“因为买的人多”，而是数学必然：波动率越高，未来股价触及行权价的概率分布越宽，期权的“时间价值”就越高。


```
卖方：
Q：卖方除了收权利金，还能以什么形式获得收入？

答案：卖方只有“权利金收入”这一种直接收入形式。

但是，这个权利金收入的来源和实现方式有两种理解角度：

卖出时一次性收到：开仓时，权利金直接进入你的账户（同时冻结保证金）。
通过“买入平仓”锁定利润：

你卖出期权时账户增加一笔钱（例如收10元）
之后如果期权价格跌到3元，你花3元买回平仓
你赚到的7元 = 收入(10) - 支出(3)
这7元本质上还是来自最初收的权利金，只是通过“低买平仓”变现
所以卖方不创造新收入形式，只是靠时间流逝 + 波动率下降让当初收的权利金变成净利润。
```


简易策略
```
环境	              波动率低	                          波动率高
股价低	            买长期看涨（便宜，等反弹）	            不买（太贵） / 可卖看跌（收贵权利金，不怕跌）
股价高	            卖看涨（收权利金，怕跌不怕涨？其实怕涨）	  买看跌（贵但可能暴跌） / 卖看涨（高风险高回报）
获利目标	          买方	                              卖方
赚方向	            是	                              否（更赚时间价值）
赚时间	            亏	                              赚
赚波动率下降	        亏	                              赚
赚波动率上升	        赚	                              亏
你提到的50%利润平仓	  买方常用	                          卖方也常用（赚权利金的一半就平仓，降低风险）
```

在期权（Options）里，ITM / ATM / OTM 是描述期权“行权价（Strike Price）”相对于标的资产当前价格位置的术语。
```
缩写	    英文	                  中文	        核心意思
ITM	    In The Money	        实值	        立即行权有价值
ATM	    At The Money	        平值	        行权价≈当前价格
OTM	    Out of The Money	    虚值	        立即行权没价值
```
```
Call
股价 vs 行权价	状态
股价 > 行权价	  ITM
股价 ≈ 行权价	  ATM
股价 < 行权价	  OTM
Put
股价 vs 行权价	状态
股价 < 行权价	  ITM
股价 ≈ 行权价	  ATM
股价 > 行权价	  OTM
```
